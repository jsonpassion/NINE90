# Building Security Update

MEMORANDUM

To: All Employees
From: Security Department
Date: March 24, 2026
Subject: Enhanced Building Security Measures

Effective April 1, the company will implement enhanced security measures across all facilities. These changes are part of our ongoing commitment to ensuring a safe working environment for all employees and visitors.

The main entrance will now require all employees to scan their ID badges upon entry and exit. Tailgating — following another person through a secured door without scanning — is strictly prohibited and will be reported as a security violation. Visitors must register at the reception desk and will be issued a temporary badge that must be returned upon departure.

Security cameras are being upgraded in all common areas, including lobbies, parking garages, and stairwells. The camera system will be monitored 24 hours a day by our security team. All recorded footage will be stored for 90 days in compliance with privacy regulations.

After-hours access to the building, between 8:00 PM and 6:00 AM, will require prior authorization from your department head. Authorized employees will receive a special after-hours access code that changes monthly. The current month's code will be distributed via secure email on the first business day of each month.

Employees who notice any security concerns should report them immediately to the Security Department at extension 9111 or security@company.com.

## Q1. What is now required at the main entrance?
- A. A fingerprint scan
- B. Scanning an ID badge upon entry and exit
- C. A password and PIN code
- D. Signing a logbook
- Answer: B

## Q2. How long is security camera footage stored?

- Answer: C

- A. Posted on the bulletin board
