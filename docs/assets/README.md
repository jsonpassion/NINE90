```html

<div class="shot-ph">📱<br />대시보드<br />스크린샷 준비 중</div>

<!-- 교체 후 -->
<img src="assets/screen-dashboard.png" alt="대시보드 화면" />
```

권장 파일명: screen-dashboard.png / screen-cards.png / screen-quiz.png
권장 크기: 1290×2796 (iPhone Pro Max) 또는 동일 비율
